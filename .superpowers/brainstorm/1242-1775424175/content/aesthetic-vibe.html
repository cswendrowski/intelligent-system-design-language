What aesthetic vibe should the default theme evoke?

The personality that comes through before any user customization

A

Parchment & Ink

Classic fantasy RPG feel. Warm tones, aged paper textures, serif-adjacent fonts. Think D&D 5e official sheets or old-school character records.

B

Modern Dark / Sleek

Dark backgrounds, glowing accents, clean lines. Think video game UI or sci-fi terminals. Works well for modern/sci-fi systems but can feel cold for fantasy.

C

Warm Neutral / Craft

Earthy, tactile feel without going full parchment. Muted warm palette, subtle textures, handmade quality. Feels like a well-loved game book. Genre-flexible.

D

Genre-Adaptive

The default should be fairly neutral but the ISDL background: parameter already sets the tone. Focus on making the existing background system carry the personality, with tighter/better-looking components on top.
