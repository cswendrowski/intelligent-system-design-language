Key Decisions

Edit Mode Signal

Subtle amber dashed border on editable fields + a small warning banner. Fields switch from display text to input boxes. Background changes to the existing cogs pattern. No heavy color shift — just enough to say "you're editing."

Container Query Scaling

Use CSS container queries (not viewport media queries) so the sheet responds to its own window size within Foundry, not the browser viewport. Grid columns, font sizes, and gaps all scale with the sheet container width.

Row Layout Preserved

The ISDL row { } construct still works — items inside a row share a horizontal line within the grid. column { } stacks items vertically within a grid cell. These map naturally to CSS grid areas.

Dark-First, Light Compatible

The design targets dark backgrounds (which Foundry defaults to), using light text and translucent field backgrounds. The rgba() approach means it still works on lighter backgrounds — the translucency adapts.
