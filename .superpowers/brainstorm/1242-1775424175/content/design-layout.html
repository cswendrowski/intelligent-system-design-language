Key Layout Changes

Quick Stats Bar

Pinned resource/status summary between header and tabs. Always visible. HP, key resources, active statuses at a glance.

No More Cards

Sections use subtle header lines + left-border accents instead of heavy card wrappers. Background shows through. Content breathes.

CSS Grid Sections

Fields flow into a 2-column grid that collapses to 1 on small sheets. Tighter than flexbox rows with min-width constraints.

Pill Tabs

Compact pill-style tab navigation instead of full-width Vuetify tabs. Saves vertical space, feels more game-UI.
