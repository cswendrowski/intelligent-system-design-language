Navigation Model (Updated)

The drawer stays — it's a proven pattern for Foundry sheets. We refine it.

Drawer: Page Navigation

Fly-out drawer keeps its role: character portrait, page selection (vertical tabs), color setup. Gets a darker, sleeker skin with pill-style page items and page icons. Closes on page select (already does this).

Horizontal Pills: Sub-tabs

Within each page, item tables/document arrays show as compact horizontal pill tabs below the quick stats bar. These replace the current horizontal Vuetify tabs that sit inside the content area.

NEW: Quick Stats Bar

A persistent bar between the app bar and content showing health resources, key stats, and active status effects. Driven by the existing health resource keyword and status properties. Always visible across all pages.

Edit Mode in Drawer

Move the edit mode toggle into the drawer's bottom section alongside color setup. Frees up the app bar and groups all "settings" actions together.
